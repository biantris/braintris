Recursividade

Acontece quando uma função pode chamar a si própria. Todo cuidado é pouco ao fazer funções recursivas. A primeira coisa a se providenciar é um critério de parada. Este vai determinar quando a função deverá parar de chamar a si mesma. Isso impede que a função se chame infinitas vezes.
